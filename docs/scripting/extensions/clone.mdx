---
title: "_CLONE"
---

A function that returns a deep-clone of a value. Any type of value may be cloned, including undefined. If the input is an object, the return value will have the same keys with cloned values for the keys.

## Syntax

```js
_CLONE(value);
```

## Arguments

### value

The value to clone.

## Example

This script wants to clone an object, so it can edit the object, while keeping it's original values:

```js
function(context, args) {
  const obj = { foo: { bar: "baz" } }
  const clone = _CLONE(obj)
  obj.foo = "baz"

  #D(obj) // {foo:"baz"}
  #D(clone) // { foo: { bar: "baz" } }
}
```
